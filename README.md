# JARVIS AI + GIF Web App

A futuristic AI named JARVIS that replies to your messages with GIFs from Tenor.

## Features
- Special replies:
  - "Who is your developer?" → Sahal Shihabudheen
  - "Which API?" → "No API — fully trained by Sahal"
- GIFs from Tenor for other queries
- Futuristic 3D animations with Three.js and GSAP
- Free hosting via Render

## How to Run Locally
```bash
npm install
node server.js
```
Open http://localhost:3000

## Deploy on Render
1. Push this repo to GitHub
2. Go to Render → New Web Service → Connect GitHub
3. Branch: main
4. Environment: Node
5. Build: `npm install`
6. Start: `node server.js`
7. Deploy & enjoy JARVIS AI live
